🚀 GitHub Alapok

Parancsok és előnyök egyszerűen

📌 Alap parancsok

Repo klónozása:

git clone URL

Változtatások hozzáadása:

git add .

Commit készítése:

git commit -m "Üzenet"

Feltöltés GitHubra:

git push

🌟 Előnyök

  • Verziókezelés (vissza tudsz lépni korábbi állapotokra)
  • Könnyű csapatmunka
  • Biztonságos felhő alapú tárolás
  • Portfólió építés fejlesztőknek

💡 Tipp

Használd a branch-eket külön fejlesztésekhez:

git checkout -b uj-funkcio
